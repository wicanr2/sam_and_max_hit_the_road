# 妙探闖通關——大腳之謎 統一譯名表

> 本表為 Phase 3 全量翻譯（6,016 行）的譯名基準。所有專有名詞翻譯**以此表為準**。
>
> **優先順序**：① 松崗官方手冊（標頁碼，一律沿用）→ ② 邱冀攻略站（當年在地資料）→ ③ 現代通行譯名 → ④ 自擬（附理由）。
>
> **來源標記**：
> - 「手冊頁 N」＝《松崗台版手冊要點整理》（`docs/manual-notes.md`），頁碼為手冊內頁碼
> - 「邱冀」＝邱冀攻略站收錄之青衫攻略，https://chiuinan.github.io/game/game/intro/eng/e21/samnmax.htm （2026-07-18 成功讀取）
> - 「現代通行」＝台灣現行通用譯名（人名、地名、影視作品等）
> - 「自擬」＝本專案自訂，備註欄附理由
>
> 名詞出處均已對照遊戲全量文本 `workplace/translation/export-en.txt`。
