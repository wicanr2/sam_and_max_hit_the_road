## 一、關於妙探闖通關（故事背景）

（手冊第 1 頁 / `doc047187_007`；盒底故事簡介 / `doc047187_002`）

大腳逃走了。這隻毛茸茸的大雪怪從馬戲團中「解凍」脫逃，同時一名女孩也跟著失蹤——是牠綁架了女孩，還是另有隱情？大腳可能躲在美國任何一個角落，也可能對當地居民造成危害。這回要靠世界頂尖的偵探搭檔**山姆**與**麥斯**上路追查，解開這樁「大腳失蹤案件」。

手冊裡是這樣鋪陳的：

- 山姆是一隻穿西裝、戴軟帽的狗偵探；麥斯是他那位瘋狂、好動、近乎失控的兔子搭檔。手冊形容他們是一對「妙搭檔」，追緝的對象包括「因為甲狀腺腫大、扛著肥脖子晃來晃去的殺手」，以及「兇惡又好吃懶做的矮胖暴徒」。
- 整起事件從**警察局長**的一通電話開始。接到電話後，山姆和麥斯便一路尖叫著開車衝進高速公路的車陣（手冊還幽默地請玩家原諒他們「非人類」的淒厲叫聲，因為車上的警笛壞了）。
- 兩人抵達 **Kushman Bros. 巡迴遊樂團**後，發現「大腳」布魯諾和「長頸女孩」翠西都失蹤了——而這只是整樁案件的開端，之後所有零碎線索會引導他們跑遍全美各地解謎。

手冊也先給新手打了預防針：如果這是你第一次玩冒險遊戲，要有點耐心，享受解謎過程；卡關時可以先去解別的謎題、找新物品，或去玩一段高速公路小遊戲換換腦袋，別放棄。

盒底的兩位主角對白也很有代表性：

- 山姆：「嘿！哥們，和你這種神經病一起被鎖在行駛中的車子裡，我想這是再輕鬆也不過的事了。」
- 麥斯：「說得太好了 Sam，我都快感動得痛哭流涕了！」

## 二、遊戲操作

### 過場動畫（cut scene）

（手冊第 3 頁 / `doc047187_008`）

遊戲一開始，山姆和麥斯呆立在辦公室裡，電話忽然響起。這段非互動、動畫式的短場景就是「**過場動畫**」，看起來像電影橋段，用來交代線索與情節（例如山姆和麥斯在 Bosco's 市場送貨那一幕）。過場動畫期間玩家無法控制人物；電話鈴聲結束後，才正式接手操控山姆的一舉一動，而麥斯會自動跟在山姆後面移動。

### 動作圖示（12 個）

（手冊第 3–5 頁 / `doc047187_008`、`doc047187_009`）

畫面下方（以及物品欄上）會有一排**動作圖示**，代表你能對場景中人物或物品做的動作。用**滑鼠右鍵**循環切換圖示、**左鍵**執行。

| # | 動作圖示 | 意義 |
|---|---|---|
| 1 | 進入／離開 物品欄 | 開關物品欄 |
| 2 | 注視；看 | 檢視目標 |
| 3 | 不能注視；不能看 | 目標無法檢視 |
| 4 | 拾起 | 拾取物品 |
| 5 | 不能拾取 | 目標無法拾取 |
| 6 | 交談 | 與人物交談 |
| 7 | 不能交談 | 目標無法交談 |
| 8 | 使用 | 使用目標 |
| 9 | 不能使用 | 目標無法使用 |
| 10 | 移動（預設動作） | 走向目標，是預設動作 |
| 11 | 站立 | 站立 |
| 12 | 使用麥斯 | 讓麥斯出手協助 |

動作圖示會**依情形自動變化**。手冊舉的例子：把「閉著的眼睛」圖示移到辦公室的電視機上，眼睛會自動張開，代表可以檢視；接著按左鍵看看會發生什麼，再試試「使用」電視會有什麼反應。

### 拾取物品

（手冊第 6 頁 / `doc047187_010`）

按滑鼠右鍵切到「拾起」圖示，再對想拿的物品按左鍵。游標會變成那樣拾起來的東西，接著你可以把它放進**物品欄**、用在物品欄裡的另一樣物品上，或用在畫面上的特定目標上。手冊的提醒：不管多誇張，只要撿得起來的東西一定有它的用處，奇怪的東西尤其別放過。

### 麥斯圖示（使用麥斯）

（手冊第 6 頁 / `doc047187_010`）

「使用麥斯」是個很特別的動作。在某些地點或對某些人物，麥斯能替山姆出一份力。把麥斯圖示移到你認為需要他協助的物品（或人物）上：

- **被紅色外框包圍**的麥斯圖示：表示這裡可以使用麥斯。
- **沒有紅色外框**：表示這裡不行。

### 與人物交談

（手冊第 6–7 頁 / `doc047187_010`）

遊戲中有許多可對談的人物，不論友善與否、幫得上忙與否，幾乎人人都有一堆話可說。在某地和某人聊完後，過一會兒再回來，對方常會告訴你新訊息——你在別處的新遭遇或發現，是引出新話題最直接的方法。

操作方式：以右鍵選「交談」圖示，把「張大嘴巴」的游標移到要交談的人物身上，等嘴巴自動張開時按左鍵即可。

### 對話圖示（4 個）

（手冊第 7 頁 / `doc047187_010`）

交談過程中，山姆主要靠畫面下方的四個對話圖示：

| # | 對話圖示 | 作用 |
|---|---|---|
| 1 | 詢問 | 提問 |
| 2 | 供述 | 陳述 |
| 3 | 閒聊 | 隨口聊 |
| 4 | 停止對話 | 結束對話 |

對話欄裡也會冒出**代表特殊項目的圖示**。手冊舉例：和某人對話時若出現布魯諾的圖示，山姆就會詢問對方有關布魯諾的事情。

### 功能鍵

（手冊第 8–9 頁 / `doc047187_011`；快速參考卡第 3 頁 / `doc047188_002`）

- **F1 或 F5**：叫出儲存／載入介面（SAVE 儲存、LOAD 載入、PLAY 回到遊戲、QUIT 中止），用來存讀遊戲進度。
- **ESC**：跳過過場動畫（或同時按下滑鼠／搖桿的兩個按鈕）。手冊笑稱：玩過好幾次之後，你大概會很需要這個鍵來略過看膩的動畫。
- **F8**：重新開始整個遊戲。
- **SPACE BAR（空白鍵）**：暫停遊戲，再按一次恢復。
- **ALT-X（或 ALT-C）**：不儲存進度、直接中止遊戲。手冊提醒：想保留進度就要記得先存檔。

### 音量與速度調整

（手冊第 8–9 頁 / `doc047187_011`；快速參考卡第 3 頁 / `doc047188_002`）

- **`[` 與 `]`**：背景音樂音量調小／調大。
- **`;` 與 `'`**：音效音量調小／調大。（若音效卡本身有音量旋鈕，記得別轉到 0 最小聲的位置。）
- **`=` 與 `-`**：文字訊息顯示速度加快／減慢。
- **`.`（DONE／關畢）**：把目前這行訊息清除掉。

### 螢幕保護

（手冊第 9 頁 / `doc047187_011`）

手冊很有戲地描述：萬一你玩到一半跑去接電話，回來卻發現整個螢幕被「小山姆和小麥斯」給侵略了，千萬別慌張——這是遊戲附贈的一堆「免費」螢幕保護功能。只要輕按一下滑鼠鍵，畫面就會恢復正常；如果沒有，那時候再慌也還來得及。

## 三、遊戲保護（換裝防拷）

（手冊第 2 頁「遊戲保護」/ `doc047187_008`）

這是台版開機時的防拷驗證，手冊是這樣說明的：

> 最初進入遊戲時，山姆和麥斯正慌張地在衣櫃裡找適當的服裝。當你按下任何一個鍵之後，就要開始替他們準備外出服。**翻到螢幕上顯示的手冊頁數**，再用箭頭圖形到衣櫃裡找出山姆或麥斯該穿的正確服裝與配件。每選好一項服飾，它會自動「神奇地」轉移到山姆或麥斯身上。全部選完後，按下螢幕正中那顆**紅色大按鈕**進入遊戲；如果弄錯了，還有第二次機會。

驗證答案就藏在手冊裡：正文第 1–10 頁**每頁頁尾都印有一幅山姆與麥斯的換裝素描**（掃描 `doc047187_007`～`doc047187_012`），遊戲要你翻到的「手冊頁數」指的就是這些內頁頁碼上的插圖穿著。

> 注意：手冊第 12 頁的「山姆和麥斯的化妝家家酒」是**遊戲內建**的換裝小遊戲（詳見第五節），雖然用同一套換裝美術，但和開機防拷是兩回事。

## 四、有用提示與善用望遠鏡

### 我們的遊戲設計理念

（手冊第 10 頁 / `doc047187_012`）

手冊特地花一段解釋設計哲學：他們相信玩家買遊戲是為了得到樂趣，而不是每次做錯事就想敲自己的頭，所以**刻意不設計會讓遊戲進死胡同或當機的機會**，避免你玩到一半莫名卡死、或因為撿了一把利器就刺死自己。他們認為玩家喜歡的是冒險與發掘祕密的過程，以及沉浸在故事裡，而不是在動詞、受詞的組合上鑽牛角尖。

### 一些有用的提示

（手冊第 10–11 頁 / `doc047187_012`）

- 只要撿得起來的物品都撿起來，再奇怪的東西也一定有用武之地。
- 卡關時先檢查身上帶的物品，想想是不是還有沒用到的地方或方法（物品欄裡可能有東西被你忽略了）。
- 再回想你去過的地方、拜訪過的人——解決問題的機會，常常就藏在你回想與新發現的物品新功能之中。

### 善用望遠鏡

（手冊第 11 頁 / `doc047187_012`）

好偵探都該懂得用望遠鏡。遊戲中你會拿到一具電子望遠鏡，永遠不知道什麼時候會用上——例如手冊提到你會跑到**溫尼倍格部落**（一個印地安村莊）去追蹤一個大壞蛋。確認望遠鏡功能正常後就試著用它：畫面下方有一根指針，移到邊緣會停住；用滑鼠左右鍵加快或減慢它的速度，左右移動指針就能改變視野方向。手冊還俏皮地補一句：「嘿嘿嘿……享受一下偷窺的樂趣吧！」

## 五、隱藏小遊戲

（手冊第 11–13 頁 / `doc047187_012`、`doc047187_013`）

因為山姆和麥斯深信「追蹤罪犯是件有趣的差事」，遊戲裡加了幾個免費、額外的袖珍小遊戲。**按 `Q` 鍵可以跳離任何一個小遊戲。**

### 打老鼠（遊樂團）

老鼠從洞裡冒出頭時，就把鐵鎚打下去。打到 20 隻以上可以獲得一項獎品。

### 高速公路大狂飆（地圖西南角）

- 滑鼠**右鍵**變換車道，**左鍵**讓麥斯跳躍。
- 盡量跳過每個車道上的招牌：每跳過一個招牌得 5 分；跳到招牌最頂端得 20 分。
- 連續跳到最頂端還有**加倍計分**的額外加分（第二個 40 分、第三個 60 分……以此類推）。
- 反過來說，沒跳過而把招牌撞碎的話，會減少麥斯的時間。

### 射擊高爾夫（地圖東南角）

有兩種玩法：

- **玩法一**：把「使用圖示」移到高爾夫球場上的每個靶上，游標會變成十字準星。任何時候只要把游標移到高爾夫球桶上按左鍵，游標就變成高爾夫旗桿；移動旗桿指到你想把球打去的地點，再按左鍵，然後按畫面上的「揮桿」按鈕。
- **玩法二**：用你自己特殊目的的**餌鳥**，把高爾夫旗桿插在這些靶的前面、指定它們游去的目標；否則整個遊戲就跟單純射擊沒兩樣、不會有變化。

### 汽車炸彈（Snuckey's 貨架）

玩法像小時候在紙上玩的戰略遊戲：

- 先點取一輛小汽車，拖曳到畫面右邊的格子上。每按一次右鍵，小汽車可 90 度旋轉一次；依此把手上其他小汽車都擺好。
- 也可以擺彈簧床，把對方的炸彈反彈回去。布好陣營、擺完自己的車，遊戲才正式開始。
- 對方的炸彈會從空中落下轟炸你的領域。選擇要丟的炸彈——**左邊普通炸彈**（威力一格）或**右邊核子彈**（威力九格）——把它們擺在你預料對方汽車可能停放的位置。一旦矇上了……砰！

### 山姆和麥斯的化妝家家酒（Snuckey's 貨架）

畫面上出現沒穿衣服、可愛的山姆和麥斯，等你替他們穿上新衣：

- 用畫面上的**紅色左右方向鍵**在不同衣櫃裡為兩人挑選服飾，選好後再用紅色左右方向鍵看看他們變成什麼德性。
- 不滿意想換，就在服飾上再點一次即可脫掉，再換上新衣服。

### 麥斯的畫冊（Snuckey's 貨架）

一套塗色遊戲：

- 按一下滑鼠鍵看整個畫面。把蠟筆頭移到顏色組上選色，再把蠟筆移到要著色的區域，按滑鼠左鍵上色。
- 要整支蠟筆搬移，就在蠟筆盒中央按一下，蠟筆變成小圖示後移到另一區域再按鍵。
- 在蠟筆盒蓋子上按鍵可跳離這個遊戲。
- 持續上色直到完成作品，替它裝個漂亮外框，然後以「高價」拿到畫廊賣掉——手冊自嘲：山姆和麥斯並不保證這來歷不明的作品有市場，事實上他們也覺得這是個笨主意。

## 六、快速參考卡重點

快速參考卡是一張折卡（六個版面），涵蓋控制、系統設定與存讀檔。

### 鍵盤／游標控制

（快速參考卡第 2 頁 / `doc047188_002`）

滑鼠右鍵切換動作圖示、左鍵執行動作或和選定物品互動。若要改用搖桿，可按 `Ctrl-M`（滑鼠）／`Ctrl-J`（搖桿）切換。沒有滑鼠時可用方向鍵／數字鍵盤（Numpad）移動游標，`Enter`／數字鍵盤 5 相當於左鍵、`Tab` 相當於右鍵。

**標準使用鍵**

| 鍵 | 對應動作 |
|---|---|
| W | Walk to 走向 |
| T | Talk to 交談 |
| U | Use 使用 |
| L | Look at 檢視 |
| P | Pick up 拾取 |
| O | 游標換成附近可拾取的物品 |
| E | 游標指在物品上時變成眼睛圖示，讓山姆檢視 |
| I | 開／關物品欄 |
| D | 往前檢視物品欄裡的物品 |
| A | 往後檢視物品欄裡的物品 |
| S | 跳到物品欄中最後一樣物品 |
| Z | 跳到物品欄中最前一樣物品 |

### 一般使用鍵

（快速參考卡第 3 頁 / `doc047188_002`）

| 鍵 | 作用 |
|---|---|
| B | 影片黑白模式切換 |
| V | 簡易動詞模式切換（動作圖示改以名稱字幕呈現）|
| Q | 跳離袖珍小遊戲（打老鼠 Wak-A-Rat、高速公路大狂飆 Hiway Surfin、汽車炸彈 CarBomb 等）|

### 特殊功能鍵

（快速參考卡第 3 頁 / `doc047188_002`）

| 鍵 | 作用 |
|---|---|
| F1 / F5 | 存讀遊戲進度 |
| ESC | 跳離過場動畫（或同按滑鼠／搖桿兩鍵）|
| F8 | 重新開始遊戲 |
| Space Bar | 暫停遊戲 |
| `[` / `]` | 背景音樂音量 小／大 |
| `;` / `'` | 音效音量 小／大 |
| `=` / `-` | 文字訊息速度 快／慢 |
| `.` | 清除文字訊息（DONE）|
| Ctrl-M / Ctrl-J | 切換滑鼠／搖桿操作 |
| Ctrl-T | 文字訊息開關 |
| Ctrl-V | 顯示遊戲版本編號 |
| ALT-X / ALT-C | 不儲存進度、中止遊戲 |

### 記憶體需求

（快速參考卡第 4 頁 / `doc047188_002`；盒底系統需求 / `doc047187_002`）

- 需要 **570K 主記憶體**；若有 2MB 以上的 EMS（Expanded Memory），主記憶體需求量可降到 **530K**。
- 主記憶體不足時遊戲開始會變慢（改讀取硬碟），實際遊玩應盡量清出足夠的主記憶體。
- 硬體平台：**IBM PC 286 以上機型，建議 386**；顯示 **256 色 VGA**；記憶體 **2MB 以上並支援 Expanded Memory**。
- 操作：搖桿／滑鼠。音效：需聲霸卡、聲霸卡專業版、AdLib 或完全相容卡；CD-ROM 版本另支援 Roland 及 General MIDI。

### 存／讀遊戲進度

（快速參考卡第 4 頁 / `doc047188_002`）

- 按 **F1 或 F5** 叫出介面，選 SAVE（儲存）、LOAD（載入）、PLAY（回到遊戲）或 QUIT（中止）。
- **儲存**：進度列在畫面左方，用上／下箭頭按鈕捲動存檔清單，選一個空位、輸入存檔名稱，按 OK（或滑鼠右鍵）存入；按 CANCEL 放棄。
- **載入**：同樣捲動清單選一個進度，按 OK；按 CANCEL 離開。
- 提醒：載入舊進度後，目前尚未存檔的進度會消失，想保留就先存起來再載入別的進度。

### 安裝與啟動

（快速參考卡第 1、5、6 頁 / `doc047188_001`）

- 建議先用 `DISKCOPY` 把原版八片磁片備份一份，保存原版、只用備份安裝。
- 安裝：DISK 1 放入 A 磁碟機（也可用 B），輸入 `INSTALL`，依畫面選音效卡（Configure Sound Board）完成設定。
- 進遊戲：切到安裝目的硬碟（例 `C:`）→ `CD SAMMNAX` → `SAMNMAX`。
- 若記憶體吃緊，參考卡「解決技術層面問題」有一段建議：做一片開機片（format `A:/S`、複製 `COMMAND.COM`、自建 `CONFIG.SYS` 設 `files=20`、`buffers=20`、自建 `AUTOEXEC.BAT` 設 `prompt`、`path`）、把滑鼠與音效驅動裝在開機片上，並用 DOS 6.0 的 **MemMaker** 最佳化記憶體、必要時解壓縮 Stacker／DoubleSpace 壓縮碟以取得最大可用主記憶體。
